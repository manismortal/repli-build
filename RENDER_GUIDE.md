# Deploying to Render

This project is configured for one-click deployment on [Render](https://render.com) using Infrastructure as Code (Blueprint).

## Prerequisites

1.  A [Render Account](https://dashboard.render.com/register).
2.  This code pushed to a GitHub repository.

## Deployment Steps

1.  **Log in to Render Dashboard.**
2.  Click **New +** and select **Blueprint**.
3.  Connect your GitHub account and select your repository (`maersk-app`).
4.  Render will detect the `render.yaml` file automatically.
5.  **Review the Plan:**
    *   It will create a **Web Service** (`maersk-app`).
    *   It will create a **PostgreSQL Database** (`maersk-db`).
6.  Click **Apply**.

## Configuration Details

*   **Build Command:** `npm run build` (Defined in `render.yaml`)
*   **Start Command:** `npm start` (Defined in `render.yaml`)
*   **Environment Variables:**
    *   `DATABASE_URL`: Automatically injected from the created database.
    *   `SESSION_SECRET`: Automatically generated by Render.
    *   `NODE_ENV`: Set to `production`.
*   **Health Check:** The app exposes `/health` which Render checks to ensure deployment success.
*   **Auto Deploy:** Enabled. Pushing to `main` will automatically redeploy.

## Post-Deployment

1.  Wait for the deployment to finish (green checkmarks).
2.  Click on the **maersk-app** service.
3.  Click the URL (e.g., `https://maersk-app.onrender.com`) to visit your live site.
